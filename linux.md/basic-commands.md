# Basic commands that we have learned today are:-

1.pwd- it stands for (print working directory) Basically it shows your location where you are right now.
2.ls- it stands for (lists) it shows all the files and directories
3.mkdir- it stands for (make directory) creates a new folder
4.cd- it stands for (change directory) navigate through diff directory
5.rmdir- it stands for (remove directory) removes an empty folder
6.touch- it  creates a new empty file
7.cp- it stands for (copy) it copies files/directories
8.rm- (remove) deletes files or directories
9.cat- (concatenate) it displays file content
10.clear- it clears the terminal screen
11.whoami- it shoes you the current logged-in user
12.echo- it prints test or writes test into a file/makes new file while writing
13.sudo- it stands for (superuser do) used to create user that can run command with admin privilege
